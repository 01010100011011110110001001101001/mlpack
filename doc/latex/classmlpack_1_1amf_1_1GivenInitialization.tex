\subsection{Detailed Description}
This initialization rule for \doxyref{A\+MF}{p.}{classmlpack_1_1amf_1_1AMF} simply fills the W and H matrices with the matrices given to the constructor of this object. 

Note that this object does not use std\+::move() during the \doxyref{Initialize()}{p.}{classmlpack_1_1amf_1_1GivenInitialization_a45da0b6c8429a47a235e04cacf35e4fb} method, so it can be reused for multiple \doxyref{A\+MF}{p.}{classmlpack_1_1amf_1_1AMF} objects, but will incur copies of the W and H matrices.
